## 手動リマインダー{#manual-reminders}

手動リマインダーは、**延滞**および**送付済み**ステータスの請求書を持つ顧客に送信できます。送信するリマインダーのメール内容をここで設定します。リマインダーの送信は、期限切れや送信済みの請求書に移動し、**その他**ドロップダウンから**リマインダーを送信**　オプションを選択することで行うことができます。

![手動リマインダー](/invoice/help/images/settings/reminder-manual.png)

リマインダーの詳細を編集するために：

* 下界では連日猛暑の**ニュースの中**、富士山頂上には雪が。
* **富士宮口登山道**を見ると、**九合目辺りまで雪**が積もっています。

![期限切れ請求書リマインダー](/invoice/help/images/settings/reminder-overdue.png)
